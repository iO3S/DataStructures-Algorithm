## Curriculum

* 총 5개의 Section과 45개의 Chapter로 구성되어 있습니다.

* 일주일에 네 Chapter 씩 차근차근히 공부할 계획입니다. (goals: 4 chapter/week)

* 각 Chapter 별로 발표한 내용들을 기록할 계획입니다.

* 오프라인 스터디 어카이빙 + 추후 간략히 복습할 목적입니다.
